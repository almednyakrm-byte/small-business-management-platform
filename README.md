# منصة إدارة الأعمال للشركات الصغيرة مع إدارة الموظفين والمنتجات
==========================

## ملخص المشروع
----------------

منصة إدارة الأعمال للشركات الصغيرة هي نظام إدارة شامل يتيح للموظفين إدارة الموارد والعمليات بشكل فعال. يحتوي النظام على إدارة الموظفين والمنتجات، مما يجعلها أداة قوية للشركات الصغيرة.

### الغرض من المشروع
------------------

* إدارة الموظفين: إدارة بيانات الموظفين، بما في ذلك معلومات التوظيف، والموارد، والصلاحيات.
* إدارة المنتجات: إدارة بيانات المنتجات، بما في ذلك معلومات المنتج، والقيمة السوقية، والتوافر.
* إدارة الأعمال: إدارة بيانات الأعمال، بما في ذلك معلومات المبيعات، والعمليات، والمراجعات.

## بنية المشروع
----------------

### الملفات والدوائر
-------------------

* `docker-compose.yml`: يحتوي على تكوين docker-compose للنظام.
* `app`: يحتوي على كود المشروع.
* `db`: يحتوي على قاعدة البيانات.
* `docker`: يحتوي على ملفات docker.

### المكونات
------------

* `app`: تطبيق الويب.
* `db`: قاعدة البيانات.
* `redis`: خادم ذاكرة الوسائط.

## إعداد البيئة
----------------

### استخدم docker-compose
-------------------------

1. تأكد من أن docker-compose مثبت على جهازك.
2. انتقل إلى مجلد المشروع.
3. استخدم الأمر التالي لتشغيل النظام:
bash
docker-compose up

4. انتظر حتى النظام يبدأ تشغيله.
5. افتح متصفح ويب واكتب `http://localhost:8000` لفتح تطبيق الويب.

## الوحدات والجداول
-------------------

### الوحدات
------------

* `users`: إدارة بيانات الموظفين.
* `products`: إدارة بيانات المنتجات.
* `sales`: إدارة بيانات المبيعات.
* `operations`: إدارة بيانات العمليات.

### الجداول
------------

* `users`: جدول بيانات الموظفين.
* `products`: جدول بيانات المنتجات.
* `sales`: جدول بيانات المبيعات.
* `operations`: جدول بيانات العمليات.

### الصلاحيات
------------

* `admin`: صلاحية الإدارة العامة.
* `user`: صلاحية الإدارة الخاصة.
* `sales`: صلاحية إدارة المبيعات.
* `operations`: صلاحية إدارة العمليات.
